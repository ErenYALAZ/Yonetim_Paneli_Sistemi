## 🖥️ About

A **Flutter Web Admin Panel** built with Dart, providing a clean and responsive management dashboard with full CRUD capabilities.

## ✨ Features

- 📊 **Dashboard** — Visual overview with stats and summaries
- 👤 **User Management** — Add, edit, delete and manage users
- 📋 **Data Tables** — Sortable and filterable record views
- 📱 **Responsive Design** — Works on desktop, tablet, and mobile

## 🚀 How to Run

```bash
# Install dependencies
flutter pub get

# Run on web
flutter run -d chrome

# Build for production
flutter build web
```

## 📁 Project Structure

